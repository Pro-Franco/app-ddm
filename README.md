# Correção de erros de execução de script no windows.
## Com o PowerShell do windows aberto faça:

1 - Veririque a politica atual

      Get-ExecutionPolicy

2 - Liberar a execução e scripts

      Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
